About this repository

General instructions

This repository contains the codes used to obtain and analyze the data used on a research study. Read about the study.

The codes used here were written using R version 4.0.3 and scripted on Quarto (.qmd) documents. Follow instruction on the Get Started page of the Quarto website to run the scripts included in this repository.

Reseach compendium

The full research compendium with data and scripts used for this study is stored in an Open Science Framework repository and can be accessed here. Download the full repository and run the analysis in your machine.
